# Simple Session Timing

A clean and simple timer I built to track my work sessions.  
You can name your session, start/pause, save your time, and reset. Perfect for productivity!

---

## Features

- Start / Pause / Resume timer
- Session naming
- "Ship it!" button to save completed sessions
- Reset button
- Nice colors taken from Hack Club's theme
- Fully responsive buttons layout
- Shows status messages while running or paused
- Sessions are saved locally
- Shows you how much you worked today
- Time and date saved for each session

---

## How to Use

1. Enter a name for your session (e.g., "Math Homework", "Coding", "Reading")
2. Click **Start** to begin the timer
3. Click **Pause** when you need a break
4. When you're done, click **✓ Ship it!** to save the session
5. Or click **Reset** if you want to start over
